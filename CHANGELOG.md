# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-28

### Added
- Initial release of Analytiq Pages Theme
- Tailwind CSS integration for responsive design
- Responsive navigation with dropdown menus
- Mobile hamburger menu
- Dark theme (Minima skin)
- Syntax highlighting with highlight.js
- Enhanced code blocks with copy functionality
- Blog layouts (post, home, page, default)
- Blog sidebar with recent posts, categories, archives
- Pagination support (jekyll-paginate-v2)
- SEO optimization with jekyll-seo-tag
- Social media links integration
- Footer with site map
- Customization hooks (custom-head, custom-header, custom-footer)
- Excalidraw integration:
  - Interactive editor layout
  - Static SVG rendering
  - Interactive iframe embedding
  - Smart embed component with multiple modes
- Tailwind components:
  - Alert component
  - Button component
  - Card component
- MathJax support for mathematical equations
- PDF embed support (jekyll-pdf-embed)
- RSS feed generation

### Theme Files
- 4 layouts: default, home, page, post, excalidraw-editor
- 16+ includes: head, header, footer, blog-sidebar, components
- Assets: CSS (Tailwind), JavaScript (pagination, Excalidraw renderer)
- Full documentation in README.md
